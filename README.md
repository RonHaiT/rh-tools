# 隆海工具箱

主要方便在前端开发中常用的工具包含：

- 日期处理
- 时间
- 身份证验证
- 表单验证

## 安装方法

使用 npm ： `npm install rh-tools`

## 身份证验证

### 验证身份证是否合法

`checkIdCard`返回值`boolean`true 是合法

### 根据身份证解析区域

`getArea`返回值`string` 如：四川

### 根据身份证解析生日

`getBirthday`返回`"yyyy-mm-dd"`

### 根据身份证解析性别

`getSex`返回值`number` 1 是男 2 是女
